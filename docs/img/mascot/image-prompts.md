# Sentinel the Fox — AI Image Generation Prompts

Pedagogical mascot for the **Cybersecurity: Foundations, Practice, and Professional Responsibility** textbook.

## Character Bible

- **Name:** Sentinel ("Sen")
- **Species:** Red fox
- **Subject:** Cybersecurity
- **Catchphrase:** "Trust, but verify."
- **Personality:** Vigilant, curious, principled, encouraging
- **Visual identity:**
    - Warm rust-orange fur with cream/white belly, throat, and tail tip
    - Black "socks" on legs and black tipped ears
    - Small round wire-rimmed glasses (subtle "scholarly" detail)
    - **Shield emblem on chest/collar** in deep cybersecurity blue (#1565c0) with a thin slate-steel border (#455a64) — never held in the hands
    - **Both hands/paws are always free and expressive** (no objects, no clipboards, no held items)
    - Friendly closed-mouth smile, alert kind eyes
    - Compact, approachable proportions; slight head-to-body ratio favoring the head for friendliness
- **Art style:** Modern flat vector cartoon, clean lines, bold but limited shading, transparent background, suitable for embedding in educational content. No text in image.
- **Color palette:**
    - Primary: deep cybersecurity blue `#1565c0` (shield, accents)
    - Secondary: slate steel `#455a64` (shield border, glasses frames)
    - Accent: alert amber `#ffa000` (use sparingly — only for warning/tip sparkles)
    - Fur: warm rust orange `#d84315` body, cream `#fff8e1` belly

> **CRITICAL DESIGN RULE:** Sentinel's hands/paws must **always remain free** so they can perform expressive gestures (waving, pointing, thinking, thumbs-up, "stop"). Do **not** put any object in the hands. The shield is a chest/collar emblem only.

> **Background rule:** Every image MUST have a fully transparent background. Do NOT use white, black, or checkered backgrounds.

## After Generating the Images

1. Save each image to `docs/img/mascot/` with the exact filenames above.
2. Required specs:
    - PNG with **fully transparent background**
    - 512×512 or 1024×1024 source resolution (will be displayed at 90px)
    - Under 100 KB per file after trimming
3. **Trim transparent padding** so the mascot fills the admonition icon space (run from project root):

```bash
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/neutral.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/welcome.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/thinking.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/tip.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/warning.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/encouraging.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/celebration.png
```

4. Run `mkdocs serve` and open the **Mascot Style Guide** page (under Learning Graph) to verify all seven poses render correctly.
